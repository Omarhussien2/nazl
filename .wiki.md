# Project Summary
This project is a web platform for downloading videos and facilitating P2P file sharing, specifically designed for users who speak the Saudi Arabic dialect. It utilizes the yt-dlp library to download content from over 1000 sites, offering a user-friendly interface. Recent updates include an advanced text transcription feature powered by AI, which converts audio from videos into written text, enhancing the platform's functionality.

# Project Module Description
The project consists of the following functional modules:
- **Frontend**: Developed with React and TypeScript, featuring a modern UI for ease of use and accessibility.
- **Backend**: Built with FastAPI for API management, integrating external services for video downloading, P2P functionalities, and audio transcription from videos.

# File Description Inventory

- **app/frontend/**: Contains frontend code, including React components and styling files.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: FastAPI, Python, httpx for HTTP requests
- **Database**: SQLAlchemy for ORM management
- **Deployment**: Vercel for hosting frontend, Atoms Cloud for backend services

# Usage
To get started with the project, follow these steps:
1. **Install Dependencies**:
   - For the backend, navigate to the `app/backend` directory and run:
     ```
     pip install -r requirements.txt
     ```
   - For the frontend, navigate to the `app/frontend` directory and run:
     ```
     pnpm install
     ```

2. **Build the Project**:
   - In the `app/frontend` directory, run:
     ```
     pnpm run build
     ```

3. **Run the Application**:
   - Start the backend server and the frontend application as per your development setup.
