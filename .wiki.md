# Project Summary
This project is a web platform designed for downloading videos and facilitating P2P file sharing, specifically tailored for users speaking the Saudi Arabic dialect. It leverages the yt-dlp library for downloading content from over 1000 sites while providing a user-friendly interface and experience. Recent enhancements include a text transcription feature that utilizes AI to convert audio from videos into written text, broadening the platform's utility.

# Project Module Description
The project consists of the following functional modules:
- **Frontend**: Built using React with TypeScript, featuring a sleek UI designed for ease of use and accessibility.
- **Backend**: Utilizes FastAPI for handling API requests, integrating with external services for video downloading and P2P functionalities. Added functionality for audio transcription from videos.

# File Description Inventory

- **app/frontend/**: Contains the frontend code, including React components, pages, and styling files.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: FastAPI, Python, httpx for HTTP requests
- **Database**: SQLAlchemy for ORM management
- **Deployment**: Vercel for hosting frontend, Atoms Cloud for backend services

# Usage
To get started with the project, follow these steps:
1. **Install Dependencies**:
   - For the backend, navigate to the `app/backend` directory and run:
     ```
     pip install -r requirements.txt
     ```
   - For the frontend, navigate to the `app/frontend` directory and run:
     ```
     pnpm install
     ```

2. **Build the Project**:
   - In the `app/frontend` directory, run:
     ```
     pnpm run build
     ```

3. **Run the Application**:
   - Start the backend server and the frontend application as per your development setup.
