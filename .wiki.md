# نزل (Nuzul) — Project Summary

Nuzul is a free, Arabic-first web platform for downloading, transcribing, and
archiving content from any public URL — video, audio, images, or carousels —
across 1,800+ platforms supported by `yt-dlp` and `gallery-dl` (YouTube,
TikTok, Instagram, Twitter/X, Pinterest, Reddit, Facebook, SoundCloud, and
more).

The product is built for Arabic podcast creators, journalists, and educators
who want fast, accurate, Arabic-dialect-aware transcripts with timestamps, and
clean downloads with no watermarks or ads.

## Modules
- **Frontend** (`app/frontend/`): React + TypeScript + Vite + Tailwind.
- **Backend** (`app/backend/`): FastAPI + SQLAlchemy. Wraps `yt-dlp` for
  video/audio and `gallery-dl` for photo galleries. Transcription pipeline
  uses Groq Whisper-Large-V3 for Arabic speech-to-text.

## Tech Stack
- **Frontend:** React, TypeScript, Tailwind, Radix UI
- **Backend:** FastAPI, SQLAlchemy, `yt-dlp`, `gallery-dl`, Groq (Whisper + Llama)
- **Database:** SQLAlchemy async (SQLite locally, Postgres/Neon in prod)
- **Deployment:** Cloudflare Pages (frontend), Cloud Run or Fly.io (backend)

## Getting Started
1. Copy env: `cp .env.example .env` and fill in the values.
2. Backend: `cd app/backend && pip install -r requirements.txt && uvicorn main:app --reload`
3. Frontend: `cd app/frontend && pnpm install && pnpm dev`
4. Or container: `docker build -t nzl . && docker run -p 8000:8000 --env-file .env nzl`
